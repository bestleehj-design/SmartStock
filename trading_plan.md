## 复盘优化规则 (2026-06-09 盘中教训)

### 规则1: 持仓分级制

执行卖出前，先给每只票打标签：

| 标签 | 定义 | 操作原则 |
|------|------|---------|
| **必清** | 趋势确认反转（如澜起-32%），不回头的品种 | 有溢价就出，不赌，不等 |
| **减仓可等** | 逻辑还在但有瑕疵（如中芯HK大基金减持） | 出一半锁风险，留一半给板块情绪空间 |
| **不动** | 核心持仓，逻辑无变化 | 无论涨跌都不动 |

### 规则2: 板块催化剂乘数（已数据验证）

SOX 涨跌对 A股半导体次日影响（2025年以来 720+ 交易日回测）：

| SOX 区间 | 样本 | 次日平均收益 | 调整乘数 |
|----------|------|------------|---------|
| SOX +5~7% | 49天 | +1.51% | 目标 × 1.015 |
| SOX +3~5% | 147天 | +1.09% | 目标 × 1.01 |
| SOX +1~3% | 700天 | +0.90% | 不调整 |
| SOX -3~-1% | 329天 | -0.25% | 不调整 |
| SOX -5~-3% | 168天 | -0.61% | 目标 × 0.99 |

**核心**: SOX 涨 5% → A股只跟 1.5%，不是直觉上的 5%。`/morning` 自动执行。

### 规则3: 反弹窗口减仓优先级

```
第1优先: 必清品种（趋势已破）
第2优先: 减仓可等品种（出半留半）
第3优先: 减仓品种
第4优先: 核心仓 → 不动
```

### 规则4: 板块催化剂 ≠ 加仓信号

```
SOX大涨 → 减仓窗口，不是加仓窗口
盘中不因「大佬买了」「XX看起来不错」临时开新仓
```

### 规则5: CPI/宏观事件前不临时加仓

宏观事件前，所有操作只减不加。事件落地后再评估。

### 规则6: 突破 vs 见顶 四维确认

| 维度 | 见顶 | 突破 |
|------|------|------|
| 成交量 | 冲高缩量 | 冲高放量 |
| 收盘位置 | 大幅回落 | 实体阳线, 收在压力上方 |
| K线形态 | 长上影线 | 实体阳线 |
| MA趋势 | MA5向下 | MA5持续向上 |

- ≥3 项突破 → 持有，设移动止盈
- 平分 → 出一半观察
- ≥3 项见顶 → 减仓

### 规则7: 「收盘判断」不能变「盘中判断」

```
✅: 收盘时检查 → 站回MA20就留
❌: 盘中看到跌就卖
假跌破恢复率 > 50% 的票 → MA20跌破等3天再决定
```

### 规则8: 最弱票 ≠ 会继续弱

反弹日弱者补涨空间最大，不要因为"同板块最弱"就卖。
