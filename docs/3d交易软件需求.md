1）策略1。先找出候选股票
1.1）量价齐升：放量定义：成交量/前若干天平均成交量 >= 1.5。目前用的是37天价格定义：涨幅大于4%
1.2）方法1：当天或者前一天5/10/30日线粘合，且当天量价齐升
1.3）方法2：当天或者前一天5/10/30日线粘合，且最近若干天放量上涨，有一天涨幅超过3%。暂时用的参数是，最近10天内，有5天放量。
1.4）所有满足策略1的票先进入备选股票池方法2的参数，可以考虑通过AI训练得到最合适的参数

2）策略2。从策略1中选出要买的票
2.1）方法1：跟随当前热点板块（例如稳定币、半导体）首先找到当前的热点板块（多个），然后找到热点板块的龙头票（TOP3），看这些票是否在策略1备选池中。如果在，则进一步进入策略2备选池。
2.1.1）热点板块获取方法：
    2.1.1.1）方法1：从tushare中获取“同花顺热榜”中的“行业板块”、“概念板块”API为 ths_hot一般是要获取连续2周的 行业或概念 的排名-----在获取某一天的排名时，会出现多个重复的，我的方法是将这几个的rank取平均值，
如：
def generate_rank_by_single_day(df):    code_to_rank = {}    result = {}
    for index, row in df.iterrows():        code = row['ts_code']        #name = row['ts_name']        rank = row['rank']        if code not in code_to_rank:            code_to_rank[code] = ( rank, 1)        else:            total_rank_count = code_to_rank[code][1] + 1            total_rank = code_to_rank[code][0] + rank            code_to_rank[code] = ( total_rank, total_rank_count)    for code in code_to_rank:        result[code] = code_to_rank[code][0] / code_to_rank[code][1]    return result-----由于某些天，某些概念或行业板块的热度排名下降，因此查询不到，就设置为100def gen_code_to_all_days_rank(hot_info):    code_to_all_days_rank = {}        for datestr in hot_info:        df = hot_info[datestr]        code_to_rank = generate_rank_by_single_day(df)        for code in code_to_rank:            rank = code_to_rank[code]            if code not in code_to_all_days_rank:                code_to_all_days_rank[code] = {}            if datestr not in code_to_all_days_rank[code]:                code_to_all_days_rank[code][datestr] = rank    for code in code_to_all_days_rank:        for datestr in hot_info:            if datestr not in code_to_all_days_rank[code]:                code_to_all_days_rank[code][datestr] = 100    return code_to_all_days_rank

    2.1.1.2）方法2：从tushare的“同花顺板块指数行情”中获取概念和板块的指数，然后根据指数的涨幅和均线等，确定当前的热门概念和行业，或者底部企稳并快速上升的概念和行业API为：ths_daily
2.1.2）热点股票获取方法：
    2.1.2.1）方法1：从tushare的“同花顺热榜”中的“热股”也要同时获取2周的数据但这些股票可能已经很热，涨幅够大，还需要结合方法2
    2.1.2.2）方法2：最近若干天最大涨幅在XX%以上的股票比如，在30天内，最高点和最低点之间的涨幅>=xx%。关注涨幅最大的TOP3或者TOP5.
2.1.3) 找到热点概念、板块和股票后，进入策略2备选池需要有一个UI界面，查看热门概念、板块和股票排名的曲线，以及该股票上涨所涉及的相关概念。并且在K线图中，可以集成查看对应的个股和概念、板块排名变化曲线。对应的参数可以定制，比如查看最近2周、4周等的数据

2.2）方法2：可能板块还没有成为热点，即根据新闻等已经提前启动对所有策略1备选池中的票，利用AI获取的板块、概念新闻以及个股新闻，分析是否有利好，并对利好进行进一步分类。主要利好包括：国家政策层面，后续大力支持发展的。例如半导体、新能源等国企改革业界改善自贸区
贸易战战争。。。
2.3）方法3：纯粹技术面交易模式分析首先利用AI，对历史出于主升浪阶段的票的起涨交易数据进行分析，然后看策略1备选池中的哪些票符合，若符合则进入备选池主要思路：获取5年内的所有股票的日线数据（如果有可能，也可以获取分钟数据），做训练数据。训练数据包括价格、成交量、资金、均线、筹码线、相关行业和概念的指数等。对于进入策略1的股票（这个不需要AI），标签数据为，短时间内（比如10天内）上涨一定幅度（比如30%以上）作为自监督数据，寻找一个模型。使得买入后，上涨大于30%的置信度大于80%。或者给出不同上涨幅度的置信度
