# 🎯 炒股选股法 — 小白版

## 🧠 核心理念（记住这一句就够了）

**只在大佬们都在的赛道里，找跑得最快的那几个选手。**

---

## 先搞懂 3 个基础概念（不然下面看不懂）

### ① 什么叫"均线"？

**就是这条街上所有人过去 N 天的平均买入成本。**

| 均线 | 通俗理解 |
|------|---------|
| **MA5** | 过去 5 天买入的人，平均成本是多少 |
| **MA10** | 过去 10 天买入的人，平均成本是多少 |
| **MA20** | 过去 20 天（约一个月）买入的人，平均成本是多少 |

**怎么看？**
- 股价在均线**上面** → 买入的人都在赚钱 ✅
- 股价在均线**下面** → 买入的人多数被套 ❌

### ② 什么叫"放量"？

**平时一天成交量 100 万手，今天突然 500 万手 = 放量。**
说明今天有大事发生，很多人冲进去买。

### ③ 什么叫"缩量"？

**平时一天成交量 100 万手，今天只有 30 万手 = 缩量。**
说明大家都不动了，观望中。

## 一、第一步：找赛道（主线识别）

### 做什么？

系统每天扫描全市场，找出**当前最热闹的板块**。

### 怎么找？

好比你在逛夜市，想知道哪家摊最火：

| 系统看的 | 换成你在夜市 |
|---------|-------------|
| 涨停占比 | 这家摊前人站了多少？ |
| 涨停梯队 | 有没有连续好几天都排长队的？ |
| 持续性 | 是天天排队还是只有第一天排队？ |
| 资金流入 | 是不是有人在真掏钱买？ |
| 成交额占比 | 整条街的注意力有多少被这家吸走了？ |

### 结果分三类

| 级别 | 意思 |
|------|------|
| ✅ **主线** | 连续好几天都是夜市最火的摊，每次路过都排长队 |
| ⚠️ **次主线** | 有时候火有时候不火，不太稳定 |
| ❌ **观察池** | 可能就火了一天，明天就没人了 |

### 举个🌰

最近 PCB（电路板）、半导体（芯片）、光模块（AI 算力）就是最火的那几桌。**沪电股份、顺络电子、长电科技都是这些桌吃饭的人。**

---

## 二、第二步：找桌里的带头大哥（龙头识别）

### 做什么？

板块选好了，接下来找这个板块里**最能打的那个**。

### 怎么找？

系统有个"大佬名单"，每条赛道谁是老大已经记好了：

| 赛道 | 带头大哥（不完整名单） |
|------|---------------------|
| 🔌 PCB | 沪电股份、深南电路、东山精密 |
| 💾 存储芯片 | 兆易创新、佰维存储 |
| 📦 芯片封测 | 长电科技、通富微电 |
| 🏭 晶圆代工 | 中芯国际 |
| 🔦 光模块 | 新易盛、中际旭创 |

### 带头大哥长什么样？

| 特征 | 通俗版 |
|------|--------|
| 连续涨停 | 其他股票涨一天就歇了，它一口气能拉好几天 |
| 换手充分 | 很多人买很多人卖，不是庄家一个人自嗨 |
| 盘子不大不小 | 50-200 亿的"微胖身材"最好：太瘦（<20亿）容易被坏人控盘，太胖（>500亿）涨不动 |

## 三、第三步：系统给每只股票打分（满分 100）

### 做什么？

系统就像考官，30 秒内给 6000 只股票挨个打分，挑出好学生。

### 加分项（做对了加分）

| 加分项 | 得分 | 用人话解释 |
|--------|------|-----------|
| **均线多头排列** | +30分 | 过去 5 天买的人 > 过去 10 天买的人 > 过去 20 天买的人 = 越晚买的人成本越高，说明**一直在涨** |
| **最近放量** | +15分 | 平时没人理的票，最近突然很多人买 |
| **属于热门赛道** | +15分 | 这条街上最火的那几家摊 |
| **是带头大哥** | +10分 | 这条赛道的龙头 |
| **有人偷偷在买** | +5分 | 成交量变大但价格没怎么涨 = 主力在悄悄吃货 |

### 扣分项（做错了扣分）

| 扣分项 | 扣分 | 用人话解释 |
|--------|------|-----------|
| 追高风险 | -10分 | 已经涨了 60%，你进去就是接盘侠 |
| 最近暴跌 | -8分 | 5 天跌了 15%，别去接飞刀 |
| 连跌 3 天以上 | -8分 | 还在跌跌不休 |
| 没人交易 | -10分 | 一天没几个人买卖，你想卖都卖不掉 |

### 最终评分

| 得分 | 评级 | 要不要买？ |
|------|------|-----------|
| **≥ 70 分** | 🟢 高分 | 多头+热门+放量，三个条件都满足，**可以考虑上车** |
| **50-69 分** | 🟡 中等 | 信号有但不完美，**再看看** |
| **< 50 分** | 🔴 低分 | 放弃吧，跳过 |

### 系统还帮你算好了"跌到哪必须跑"

```
止损价 = min(MA20, 近10天最低价) × 0.97
```

就是：**跌破这条线立刻走人，不犹豫。**

## 四、第四步：你来做人工检查（系统做不到的 3 件事）

系统打分再高，也有 3 件事做不了，需要你过目：

### ① 有政策支持吗？

```
系统：这只票技术面 85 分！
你：等等，这板块有国家政策支持吗？
```

| 结果 | 判断 |
|------|------|
| 国家大基金投了半导体 | ✅ 可以买 |
| 没人管没人问的冷门板块 | ❌ 再想想 |

### ② 公司真的赚钱吗？

```
系统：这只票最近涨得很猛！
你：等等，它赚的钱是真的吗？
```

| 结果 | 判断 |
|------|------|
| 业绩爆发，订单排到明年 | ✅ 真的在赚钱 |
| 纯概念炒作，没业绩 | ❌ 可能会跌回去 |

### ③ 新闻是好事还是坏事？

| 新闻 | 判断 |
|------|------|
| 中标大订单 | ✅ 利好 |
| 大股东要卖股票 | 🔴 快跑！老板自己都在卖 |
| 被证监会问话了 | 🔴 快跑！可能有问题 |
| 尾盘快收盘突然拉涨停 | 🔴 不是真买盘，是忽悠人接盘 |

### 一票否决（不管多少分，直接淘汰）

| 情况 | 怎么办 |
|------|--------|
| 这个板块就火了一天，第二天就没人提了 | ❌ 一日游，不碰 |
| 大股东在偷偷卖股票 | ❌ 老板都在卖，你买？ |
| 20 天涨了 100% 了 | ❌ 太贵了，进去就是接盘 |
| 公司被 ST 了（有退市风险） | ❌ 别碰，会死的 |

## 五、第五步：买多少？（仓位管理）

### 系统帮你算好该买多少

```
你应该买的金额 = 总资金 × 推荐仓位 %
```

### 而且还帮你分 3 批买

```
假如系统让你买 10 万块：

  第 1 批：5 万（试探性买入）
    → 先买一半看看
    
  第 2 批：3 万（跌了再加）
    → 如果跌了但没破止损线，再买一点摊低成本
    
  第 3 批：2 万（涨了追加）
    → 如果确定涨上去了，再加最后一把
```

### 铁律

**一只股票最多买总资金的 25%。** 不 all in，不赌命。
