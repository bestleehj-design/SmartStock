# 🎯 选股方法 — 小白版

> 把复杂的量化选股方法翻译成人话。

---

## 一句话总结

**只在大佬们都在的赛道里，找跑得最快的那几个选手，让系统帮你打分验证。**

---

## 🧠 核心思想（就一条铁律）

> **不在主线上的股票，再好也不碰。**

什么叫主线？就是最近资金都在追捧的板块——好比**酒吧里最热闹的那一桌**，大家都往那儿挤，酒水自然有人买单。

冷门票就像空荡荡的角落卡座，气氛组都不去，你去干嘛？

---

## 🏗️ 整套方法分 5 步（从大到小）

---

### 第 1 步：找赛道（主线识别）

系统每天扫描全市场，用 6 个维度给每个概念板块打分：

| 打分维度 | 通俗版 |
|---------|--------|
| 涨停占比 | 这桌有多少人站起来了？ |
| 涨停梯队 | 有没有连续几天站起来的"大哥"带队？ |
| 持续性 | 是天天嗨还是只嗨了一天？ |
| 资金流入 | 真金白银在往里冲吗？ |
| 板块涨幅 | 这桌人整体赚了多少？ |
| 成交额占比 | 全场的注意力有多少在这桌？ |

**结果就三类：**

| 等级 | 含义 |
|------|------|
| ✅ **第一梯队（主线）** | 连续几天都在 Top 3，持续热闹 |
| ⚠️ **第二梯队（次主线）** | 偶尔热闹，但不够持久 |
| ❌ **观察池** | 可能就嗨了一天，看看再说 |

**举个🌰：** PCB、半导体、光模块就是第一梯队的"嗨桌"。沪电股份、顺络电子、长电科技都是这桌的人。

---

### 第 2 步：找桌里的头部玩家（龙头识别）

赛道选好了，接下来要找这个板块里**最靓的那个仔**。

系统有个内部"白名单"，好比**每桌固定的大佬名单**：

| 桌 | 大佬 |
|----|------|
| PCB 桌 | 沪电股份、深南电路、东山精密 |
| 存储芯片桌 | 兆易创新、佰维存储 |
| 封测桌 | 长电科技、通富微电 |
| 晶圆代工桌 | 中芯国际 |
| 光模块/CPO 桌 | 新易盛、中际旭创、天孚通信 |

这些大佬通常满足三个条件：
1. **连续涨停** — 别的票涨一天就歇了，它能连拉三四天
2. **换手充分** — 有人买有人卖，筹码交换活跃，不是庄家自嗨
3. **盘子大小适中** — 50-200 亿最好的"微胖"身材：太瘦（20亿以下）容易被游资控盘，太胖（500亿以上）涨不动

---

### 第 3 步：给每只股票打分（Smart Screener）

这是系统的核心——**30 秒扫完 6000 只股票**，给每只打 0-100 分。

#### ✅ 加分项（好孩子加分）

| 加分项 | 分值 | 通俗版 |
|--------|------|--------|
| 均线多头 | +30 | MA5 > MA10 > MA20，三条线从低到高排好队 = 趋势向上 |
| 放量 | +15 | 最近成交量比之前大 = 有人在买 |
| 在主线板块 | +15 | 属于上面说的热门赛道 |
| 是龙头 | +10 | 白名单里的带头大哥 |
| 底部吸筹 | +5 | 偷偷放量但没涨 = 主力在悄悄吃货 |

#### ❌ 扣分项（坏孩子扣分）

| 扣分项 | 分值 | 通俗版 |
|--------|------|--------|
| 追高风险 | -10 | 已经涨了 60% 了，小心接盘 |
| 短期暴跌 | -8 | 5 天跌了 15%，别接飞刀 |
| 连跌 3 天 | -8 | 跌跌不休，先别碰 |
| 流动性太差 | -10 | 一天没几个人交易，想卖都卖不掉 |

#### 打分结果直接看等级

| 得分 | 评级 | 意思 |
|------|------|------|
| **≥ 70 分** | ✅ 高分 | 多头+主线+放量，三个信号都亮，可以上车 |
| **50-69 分** | ⚠️ 中等 | 信号有但不完美，再查查 |
| **< 50 分** | ❌ 低分 | 垃圾时间，跳过 |

#### 系统还会自动算止损价

```
止损价 = min(MA20, 最近10天最低价) × 0.97
```

机器帮你想好了：**跌破这个价就认输走人**。

---

### 第 4 步：人工验证（机器不能做的 3 件事）

系统再聪明，也有 3 件事做不了，得你来判断：

| 系统不行的 | 你来问 | 好例子 | 坏例子 |
|-----------|--------|--------|--------|
| ❌ **政策面** | 这板块有国家政策支持吗？ | 半导体有国家大基金 = ✅ | 纯散户炒作 = ❌ |
| ❌ **营收前景** | 公司赚的钱是真的还是吹的？ | 新易盛业绩爆发 = ✅ | 概念股没业绩 = ❌ |
| ❌ **新闻解读** | 这新闻是利好还是雷？ | 中标大订单 = ✅ | 大股东减持 = 🔴 |

#### 一票否决（不管多少分，直接滚）

| 情况 | 原因 |
|------|------|
| 板块就嗨了一天，明天就凉 | 一日游，不碰 |
| 公司大股东在偷偷卖股票 | 内部人都在跑，你进？ |
| 尾盘突然拉涨停 | 不是真买盘，是忽悠，不碰 |
| 20 天涨了 100% | 太贵了，纯接盘 |

---

### 第 5 步：买多少？怎么买？（仓位管理）

系统用**凯利公式**算你应该买多少：

```
仓位 = (胜率 × 赔率 - 输的概率) ÷ 赔率
```

**看着晕？系统自动帮你算好了，还给你分了 3 批买入：**

```
假如系统建议你买 100 块：

  第 1 批：50 块（试探）
    → 先买一半看看
    
  第 2 批：30 块（回踩加仓）
    → 如果跌了但没破止损，再加
    
  第 3 批：20 块（突破追加）
    → 如果真的涨上去了，再加最后一把
```

**铁律：单只股票最多买总资金的 25%**，不 all in，留条活路。

市场情绪也会影响仓位：

| 市场状态 | 仓位系数 | 意思 |
|---------|---------|------|
| 😱 恐慌日（百股跌停） | 1.3 | 别人恐惧我贪婪，可以多买点 |
| 🥳 亢奋日（百股涨停） | 0.5 | 太嗨了，少买点，小心回调 |
| 😐 中性 | 1.0 | 正常买 |
