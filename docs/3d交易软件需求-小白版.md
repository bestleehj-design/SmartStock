## 第一步：策略1 — 找"有动静"的股票

### 先搞懂 2 个基础概念

#### ① 什么是"均线"？

**就是不同时间段的平均买入价格。**

| 名称 | 等于 | 可以理解为 |
|------|------|-----------|
| **MA5** | 过去 5 天的平均价 | 这周买的人平均花了多少钱 |
| **MA10** | 过去 10 天的平均价 | 最近半个月买的人平均花了多少钱 |
| **MA30** | 过去 30 天的平均价 | 最近一个月买的人平均花了多少钱 |

#### ② 什么是"均线粘合"？

```
正常情况（均线散开）：
  MA5  ───────── 10 元  ← 最近买入的人成本高
  MA10 ──────── 9 元
  MA30 ──────── 8 元  ← 一个月前买入的人成本低
  
  → 说明股票一直在涨，价格层层抬高

粘合情况（三条线快缠一起了）：
  MA5  ──────── 9.4 元
  MA10 ──────── 9.3 元  ← 三者相差不到 4%
  MA30 ──────── 9.1 元
  
  → 说明最近一个月买入的人成本都差不多
  → 股票横盘了很久，没涨也没跌
  → 像一根被压紧的弹簧，随时可能弹起来
```

#### ③ 什么是"放量"？

| 平时（正常） | 今天（放量） |
|------------|------------|
| 每天 100 万人买卖这只股票 | 突然变成 500 万人买卖 |
| 没人关注，风平浪静 | 🔥 出大事了！ |

**放量 = 有人在大动作。**

---

### 策略 1 在找什么？

**找"均线粘合 + 放量上涨"的股票。**

用人话翻译就是：

> **一只股票横盘不动很久了（像冬眠），今天突然很多人冲进去买，价格开始涨了（像睡醒了）。这种票可能就是一波大涨的起点。**

### 怎么筛？

两种方法，满足任一就算通过：

| 方法 | 条件 | 大白话版 |
|------|------|----------|
| **方法 1** | 5/10/30 日均线粘合 + 今天突然放量涨了 4% 以上 | 这只股票横盘很久了，今天突然有人大量买入，价格拉起来了 |
| **方法 2** | 均线粘合 + 最近 10 天里有 5 天都在放量上涨 | 这只股票横盘很久了，最近有人一直在偷偷买，慢慢建仓 |

**任何满足条件 → 进候选池 📋（大概 200 只）**

## 第二步：策略2 — 找"值得买"的股票

从 200 只里精选。有三种方法，**任何一种通过就能进下一轮。**

### 方法 1：跟热点 ✅（最推荐）

**先看今天最火的板块 → 板块里的带头大哥 → 大哥有没有在候选池里**

> 通俗理解：先看今晚酒吧哪几桌最热闹 → 看每桌谁是老大 → 看老大是不是刚才策略 1 那个"有动静"的人

#### 怎么知道哪个板块最火？

同花顺每天给每个板块排名，像微博热搜一样：

```
今日板块热搜榜 🏆
  🥇 半导体   ← 连续 2 周都在前排 = 真热点
  🥈 AI 算力  
  🥉 储能
```

#### 怎么找板块里的带头大哥？

| 方法 | 怎么找 |
|------|--------|
| 人气最高的 | 同花顺热股榜上排前面的 |
| 涨得最猛的 | 最近一段时间涨最多的前几名 |

### 方法 2：提前埋伏（找还没火起来的）

**热门板块已经被很多人知道了，但如果能提前知道下一个热门呢？**

系统用 AI 天天看新闻，如果有板块还没火但新闻已经在吹了，系统会发现：

```
新闻："国家出台新政策，大力支持储能产业发展！"
     ↓
系统自动搜索候选池里有没有做储能的股票
     ↓
如果有，这些股票自动进下一轮 ✅
```

| 利好类型 | 例子 |
|---------|------|
| 🏛️ **国家政策** | "国家大基金三期成立" → 半导体要涨 |
| 🏭 **行业涨价** | "芯片全面涨价" → 芯片股要赚更多钱 |
| 🔄 **公司重组** | "国企混改" → 可能要变好 |

### 方法 3：AI 预测（让 AI 告诉你哪只可能会暴涨）

> **这个比较烧脑，我讲简单点。**

让 AI 去学习**过去 5 年所有股票的暴涨历史**：

```
AI 的学习过程 📚
  第 1 课：看过去 5 年每只股票什么时候暴涨过
  第 2 课：分析它们暴涨前有什么共同特征
          → 成交量变大了？
          → 均线粘合了？
          → 板块在升温了？
  第 3 课：用学到的规律扫描现在的候选池
  第 4 课：告诉你"这只票有 85% 的概率会涨"
```

可以理解为：**让 AI 看了一万场足球比赛录像，学会了哪些配合能进球，然后告诉你现在场上哪个队要进球了。**

## 第三步：策略3 — 你做最终决定

AI 帮你查好了每只候选股的"身家背景"，你来做最后的判断。

### AI 会帮你整理好这些

```
📋 股票简历

  1. 基本信息：这公司是干啥的？（卖芯片的？还是卖酱油的？）
  2. 财务体检：近几年赚钱了吗？是越来越好还是越来越差？
  3. 新闻汇总：最近有没有好消息（中标了）？有没有坏消息（被罚款了）？
  4. 公告检查：有没有潜在的地雷？
```

### 你的加分清单

看完资料后，满足以下条件的加分：

| 情况 | 打勾 | 说明 |
|------|------|------|
| ✅ 日线、周线、月线都多头 | ☐ | 无论短期、中期、长期趋势都向上 |
| ✅ 横盘很久了 | ☐ | 粘合时间越长，爆发力越强 |
| ✅ 放量但没大涨 | ☐ | 有人在偷偷买，还没开始拉 |
| ✅ 小碎步涨 | ☐ | 每天涨一点点，不回跌，很稳 |
| ✅ 涨的时候放量，跌的时候缩量 | ☐ | 健康的上涨姿势 |

### 你的否决清单（有一条就淘汰）

| 情况 | 直接 |
|------|------|
| 🔴 大股东在卖股票 | 老板都跑路了，你进去？ |
| 🔴 被证监会问话了 | 可能有雷，别碰 |
| 🔴 业绩预告说亏了很多钱 | 基本面烂了 |
| 🔴 公司根本不是做这个的 | 纯蹭概念，跌回去是迟早的事 |

## 📊 和 Smart Screener 的区别

```
Smart Screener（追涨派）
  👉 找已经涨起来的强势股，继续追
  👉 好比看到一匹马赢了第一场，买它第二场继续赢

3D 策略（潜伏派）
  👉 找横盘很久快要启动的，提前进去等涨
  👉 好比看一匹马状态很好、在起跑线热身，赌它马上要冲第一

Smart Screener 是 "追已经跑起来的"
3D 策略是       "赌马上要跑的"
```
