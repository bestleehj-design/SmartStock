# 超跌反弹选股器 — 设计文档 v2

> 回测验证: 2022-11 ~ 2026-06，9 天同步大跌日，45,822 条股票-日记录

---

## 1. 回测结论（先看结果，再看设计）

### 大盘同步大跌日，次日全市场平均反弹 +0.68%，胜率 60%

博反弹的底层逻辑成立。但**只有 2 个因子有效**：

| 因子 | 回测结论 | 数据 |
|------|---------|------|
| **跌幅深度** ✅ | 核心因子，-5~-8% 最强 | 次日 **+2.30%, 胜率 82%** |
| **MA60 位置** ✅ | 辅助因子 | MA60上方 +1.03% vs 下方 +0.32% |
| 盘中承接（长下影） | ❌ 无效 | 有承接 +0.80% vs 无 +0.69%，差 0.11% |
| 板块主线 | ❌ 反向 | 主线 +0.50% vs 线外 +0.71%，主线更差 |

### 跌幅区间次日表现

```
抗跌(+0%+)   -0.15%  48%  → 不跌不需要弹
小跌(-1~3%)  -0.28%  44%  → 太浅没弹性
中跌(-3~5%)  +0.91%  66%  → 可以
深跌(-5~8%)  +2.30%  82%  ← 黄金区间
崩盘(-8%+)   +0.61%  61%  → 太狠是真崩
```

## 2. 评分模型（只有 2 个因子，总分 100）

### 核心因子：跌幅深度（60 分）

| 跌幅 | 得分 | 依据 |
|------|------|------|
| 5% ~ 8% | **60** | 回测 +2.30%, 胜率 82% |
| 3% ~ 5% | **30** | 回测 +0.91%, 胜率 66% |
| 1% ~ 3% | **0** | 回测 -0.28%, 无弹性 |
| > 8% | **0** | 回测 +0.61%, 胜率仅 61%，且可能是真崩 |
| < 1%（抗跌） | **0** | 不需要反弹 |

### 辅助因子：MA60 过滤（40 分）

| 条件 | 得分 |
|------|------|
| `收盘 > MA60` | **40** |
| `收盘 < MA60` | **0（一票否决）** |

### 为什么这么简单？

K 线形态不可靠（量化制造下影线诱多）、板块主线反向（大跌日砸的就是主线）、RSI 没有区分力。两个因子已经够用。

## 7. 设计原则（从教训中学到）

1. **先回测再编码** — 5 个模块砍成 2 个因子，省了 200 行无用代码
2. **简单优于全面** — 15 个因子和四维评分一样全盘无效，不如两个验证过的因子
3. **量化环境下放弃 K 线形态** — 长下影、V 型反弹数据说话不靠谱
4. **大跌日不要选主线票** — 回测显示主线反弹更差，砸的就是热门
