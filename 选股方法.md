## 一、总体框架

本系统采用**自上而下**的选股逻辑：先定主线方向，再在主线内找强势个股，最后结合技术面和基本面评分排序。

```
主线识别 → 龙头定位 → 个股评分 → 深度验证 → 持仓监控 → 回溯优化
```

**核心理念：只做主线上的强势票，不在主线上的绝不碰。**

## 二、主线题材识别

### 2.1 数据来源

`theme_analyzer.py` 通过 tushare 导入的同花顺概念板块数据，对全市场概念板块进行评分。

- 数据采集：`new_get_all_stock.py` 通过 tushare `ths_index` + `ths_member` API 拉取并入库
- 概念板块来源：`stock_basic_info_tbl` 表中 `type=2` 的概念记录
- 成分股映射：同花顺概念板块的成分股列表
- 仅分析成分股 ≥ 10 只的板块（排除微型板块）

### 2.2 六维评分体系

每个概念板块从 6 个维度独立打分（每维 0-10 分），总分满分 60 分：

| 维度 | 分值 | 评分逻辑 |
|------|------|---------|
| **涨停占比** | 0-10 | 涨停股数 / 板块总股数，10% 以上满分 |
| **涨停梯队** | 0-10 | 高标(≥3连板) 每只3分上限5 + 中位(2连板) 每只2分上限3 + 首板每只0.5分上限2 |
| **持续性** | 0-10 | 近5日涨停活跃度趋势，连续增长额外+2分 |
| **资金流入** | 0-10 | 板块大单净流入占板块成交额比例，2%净流入满分 |
| **板块涨幅** | 0-10 | 成分股近5日平均涨幅(权重0.6) + 近10日平均涨幅(权重0.4)，涨幅越大越高 |
| **成交额占比** | 0-10 | 板块成交额 / 全市场成交额，4%以上满分 |

### 2.3 过滤规则

自动排除以下非真正题材的概念：

- **宏观大盘指数**：同花顺全A、同花顺沪深、同花顺大市值等
- **融资/资金类**：融资融券、深股通、沪股通
- **打板追踪指标**：昨日涨停、昨日首板、龙虎榜指数
- **业绩/财务风格类**：业绩预亏、低市盈率
- **地域类**：粤港澳大湾区、京津冀等
- **宽基指数**：国企改革（太宽泛）

### 2.4 主线筛选标准

从日度分析结果中自动识别主线：
- 总分 ≥ 20 分
- 取 **Top 3** 作为当日主线题材

### 2.5 长期主线分析模式

`analyze_long_term_themes()` 方法从数据库中读取近 20 个交易日的历史评分，计算四个长期指标：

| 指标 | 权重 | 说明 |
|------|------|------|
| **累计强度** | 35% | 期间总分累计，越高说明炒作越持续 |
| **日均强度** | 25% | 有活动的天数平均分，排除一日游 |
| **登顶频率** | 20% | 进入 Top5 的天数占比，频次高说明持续领涨 |
| **趋势方向** | 20% | 评分线性回归斜率，上升(/)、稳定(→)、下降(\) |

长期筛选条件：
- **常规长期主线**：活跃 ≥ 5 天 + 日均分 ≥ 10 + 趋势非下降
- **新兴主线**：活跃 ≥ 3 天 + 趋势上升 + 日均分 ≥ 10

### 2.6 当前关注的 8 条主线板块

| 主线 | 跟踪关键词 |
|------|-----------|
| 光模块/CPO | 光模块、光通信、CPO、800G、1.6T、光器件 |
| PCB | PCB、印制电路板、覆铜板 |
| 电感/被动元件 | 电感、被动元件、MLCC、电阻、电容 |
| AI 服务器 | 服务器、算力、液冷、数据中心 |
| 存储芯片 | 存储、HBM、DDR5、DRAM、NAND |
| 半导体封测 | 封测、封装、Chiplet、先进封装 |
| 半导体设备 | 刻蚀、薄膜、光刻 |
| 晶圆代工 | 晶圆、代工、Foundry |

### 2.7 用法

```bash
# 分析最新交易日主线题材
python3 theme_analyzer.py

# 分析指定日期
python3 theme_analyzer.py 2026-06-05
```

## 三、龙头票识别

`leader_analyzer.py` 在已识别的主线题材内，通过四维评分选出龙头票。

### 3.1 龙头票四维评分

| 维度 | 满分 | 评分逻辑 |
|------|------|---------|
| **连板高度** | 40 | 连续涨停天数 × 10，4连板以上满分 |
| **量能强度** | 30 | 换手率 / 2，换手率 60% 满分（涨停板换手充分说明筹码健康） |
| **流通市值** | 15 | 50-200亿(15分) > 20-50亿(12分) > 200-500亿(10分) |
| **资金强度** | 15 | 大单+超大单净买入占成交额比，3%以上满分 |

**前提条件：必须是当日涨停的股票。**

### 3.2 市值偏好说明

- **50-200 亿流通市值**最优：盘子适中，有上涨空间，资金能进出
- **20 亿以下**：小票，流动性差，游资控盘风险
- **500 亿以上**：大盘股，弹性不足，上涨慢

### 3.3 龙头白名单

系统维护了一份手工白名单 `LEADER_WHITELIST`，对应每条主线的 A 股龙头代码：

| 主线 | 白名单代码 |
|------|-----------|
| 光模块/CPO | 300502(新易盛), 300308(中际旭创), 300394(天孚通信), 300570(太辰光), 688498(源杰科技) |
| PCB | 002463(沪电股份), 002916(深南电路), 300476(胜宏科技), 002384(东山精密), 603228(景旺电子) |
| 电感/被动元件 | 002138(顺络电子), 300408(三环集团), 603678(火炬电子) |
| AI 服务器 | 601138(工业富联), 000938(紫光股份), 300502, 688041, 603019 |
| 存储芯片 | 603986(兆易创新), 688525(佰维存储), 001309, 688981 |
| 半导体封测 | 600584(长电科技), 002156(通富微电), 603005(晶方科技), 688981 |
| 半导体设备 | 002371(北方华创), 688012(中微公司), 688072, 688120, 688082 |
| 晶圆代工 | 688981(中芯国际), 688347, 688396 |

### 3.4 行业纯度校验

股票必须通过申万行业前缀校验才能被归入某板块，防止名不副实的股票混入。

| 主线 | 允许的申万行业前缀 |
|------|------------------|
| 光模块/CPO | 通信、电子 |
| PCB / 电感 / 存储 / 封测 | 电子 |
| AI 服务器 | 电子、计算机、通信 |
| 半导体设备 | 电子、机械设备 |
| 晶圆代工 | 电子 |

### 3.5 用法

```bash
# 自动运行主线分析 + 龙头识别
python3 leader_analyzer.py

# 指定日期
python3 leader_analyzer.py 2026-06-05
```

## 四、Smart Screener 综合评分选股

`smart_screener.py` 是核心选股引擎，对全市场 A 股（约 6120 只）进行扫描评分，30 秒出结果。

### 4.1 扫描范围

- **默认**：仅在 8 条主线的成分股中扫描（约 1500 只）
- `--all`：全市场扫描
- `--sector PCB`：指定板块扫描

### 4.2 前置过滤

评分前自动过滤：
- **日成交量**：近 20 日均量 < 50 万手的股票（流动性太差的冷门票直接排除）
- **板块过滤**：默认只在主线板块成分股中扫描

### 4.3 多维度评分体系（满分 100）

#### 加分项

| 维度 | 满分 | 评分细则 |
|------|------|---------|
| **均线多头** | 30 | MA5 > MA10 > MA20 多头排列 = +20 / 短多(MA5>MA20) = +12 / 站上MA20 = +6<br>距MA20在 (0, 10%) = +10 / 回踩MA20 (-3%, 0) = +8 |
| **成交量** | 15 | 近5日均量 > 近前5日均量 × 1.3 放量 = +10 / > 0.8 温和 = +5 |
| **题材归属** | 15 | 在主线板块 = +15 / 非主线 = +2 |
| **龙头权重** | 10 | 白名单龙头票 = +10 / 主线板块成分股 = +2 |
| **市值级别** | 5 | 大盘(>100元) = +5 / 中盘(>30元) = +3 |
| **底部吸筹** | 5 | 近10天内≥5天满足：量>37日均量×1.5 且当日涨跌幅在 (-3%, 2%) 区间（放量但不涨，主力建仓信号） |
| **碎步小阳** | 5 | 近10天内≥7天涨幅在 (0, 3%)（多小幅持续上涨，蚂蚁上树形态） |
| **恐慌日抗跌** | 25 | （仅恐慌模式）翻红 > 1% = +25 / 平收 > -2% = +18 / 跌超5% = +2 |

#### 扣分项

| 风险项 | 扣分 | 触发条件 |
|--------|------|---------|
| 追高风险 | -10 | 20日涨幅 > 60% |
| 短期暴跌 | -8 | 5日跌幅 < -15% |
| 连续下跌 | -8 | 连跌 ≥ 3天 |
| 流动性差 | -10 | 近5日均量 < 50万手 |
| 中期涨幅过大 | -3 | 20日涨幅 > 30% |
| 短期走弱 | -3 | 5日跌幅 < -8% |

#### 深度模式额外扣分（`--deep`）

对 Top 结果拉取新闻进行验证：
- 发现**公司澄清**、**不存在业务** → -15 分
- 发现**立案**、**处罚**、**违规** → -15 分

### 4.4 风险等级

| 得分 | 评级 | 含义 |
|------|------|------|
| ≥ 70 | 高分 | 多头排列 + 主线 + 放量，信号共振强劲 |
| 50-69 | 中等 | 有正面信号但不够完美，需进一步验证 |
| < 50 | 低分 | 信号弱或有明显风险项 |

### 4.5 止损计算

每只股票自动计算建议止损价位：

```
止损价 = min(MA20, 近10日最低价) × 0.97
风险% = (现价 - 止损价) / 现价 × 100%
```

### 4.6 输出格式

```
#1  顺络电子 (002138)  得分: 82 ⭐龙头
    行业: 电子  板块: 电感/被动元件  现价: 35.20
    涨跌: +2.50%  成交额: 8.5亿
    加分: 多头排列 | MA20 +5% | 放量 | 主线: 电感/被动元件 | ⭐龙头票 | 中盘
    止损: 31.50  风险: 10.5%
```

### 4.7 用法

```bash
# 基础筛选（主线板块 + TOP20）
python3 smart_screener.py

# 恐慌日模式（优先选抗跌翻红票）
python3 smart_screener.py --panic --top 10

# 指定板块
python3 smart_screener.py --sector PCB --top 10

# 深度模式（拉新闻排除雷）
python3 smart_screener.py --deep --top 10

# 全市场扫描
python3 smart_screener.py --all --top 20

# 保存结果到数据库
python3 smart_screener.py --save

# 回填历史收益（用于回溯分析）
python3 smart_screener.py --backfill

# 分析历史筛选效果
python3 smart_screener.py --analyze
```

## 五、题材日历——季节性题材加权

`sector_calendar.py` 根据当前日期自动识别处于炒作窗口的季节性题材，在 Smart Screener 评分中给予加权。

### 5.1 设计逻辑

- 每段时间区间对应一个题材（如 6-8 月 → 电力）
- 提前 **2 周**进入预热期，线性增长至满分
- 末周进入退潮期，线性衰减
- 返回板块关键词和 boost 系数，供 `smart_screener.py` 加权

### 5.2 题材日历配置

| 时间 | 题材 | 关键词 | 逻辑 |
|------|------|--------|------|
| 6/1 - 8/31 | 夏季用电高峰 | 电力、电网、智能电网、虚拟电厂、特高压、风电 | 夏季用电负荷创新高 |
| 11/15 - 3/15 | 冬季取暖 | 燃气、煤炭、供热、天然气 | 冬季取暖季需求上升 |
| 1/10 - 2/15 | 春节消费 | 白酒、食品饮料、预制菜、旅游、影视 | 春节假期消费旺季 |
| 4/20 - 5/10 | 五一小长假 | 旅游、酒店、餐饮、家电 | 五一假期消费 |
| 9/15 - 10/10 | 十一黄金周 | 旅游、酒店、消费、零售 | 十一黄金周消费 |
| 3/1 - 4/30 | 春耕 | 种业、化肥、农药、农机 | 春耕季节农资需求 |

### 5.3 在 Smart Screener 中的集成

- 默认开启：自动识别活跃的季节性窗口
- 加权幅度：最多 +8 分（`boost × 8`），预热/退潮期按比例递减
- 匹配方式：股票名称中包含季节性关键词的，给予对应加权
- 关闭方式：`--no-seasonal` 禁用

### 5.4 用法

```bash

python3 sector_calendar.py

# 查看指定日期
python3 sector_calendar.py 2026-07-15

# 在筛选中关闭季节性加权
python3 smart_screener.py --no-seasonal
```

## 六、人工判断环节

系统自动化评分覆盖了**技术面 + 资金面 + 主题归属 + 季节性**，但以下维度必须人工验证。

### 6.1 系统能做的 vs 不能做的

| 维度 | 自动化 | 人工介入点 |
|------|--------|-----------|
| 技术面 | ✅ 均线、量能、MACD、RSI、涨停分析 | 判断死叉是洗盘还是真转弱 |
| 资金面 | ✅ 大单净额、成交额占比、换手率 | 判断主力是建仓还是出货 |
| 主题归属 | ✅ 关键词+行业纯度校验 | 判断板块逻辑是否成立 |
| 季节性 | ✅ 题材日历自动加权 | 突发事件可能比季节性更强 |
| 政策面 | ❌ | **必须人工**：有无国家政策/大基金/产业规划支持 |
| 营收前景 | ❌ | **必须人工**：订单可见度、行业渗透率、下游需求 |
| 新闻深度解读 | ⚠️ 关键词匹配 | **必须人工**：减持公告、监管问询、诉讼风险 |

### 6.2 逐票 Checklist

拿到 Smart Screener 的 Top 20 后，对每只候选票依次检查：

```
[ ] 所在板块是否有明确的政策/产业逻辑支撑？
[ ] 板块近期涨停梯队是否完整（高标 ≥ 中位 + 首板）？
[ ] 公司是否是板块内的核心标的（龙头白名单或行业校验通过）？
[ ] 近期有无减持公告、监管问询、业绩暴雷？
[ ] 今日分时：主动性买盘还是被动跟涨？尾盘是否回落？
[ ] 20 日涨幅是否过高（> 60% 追高风险）？
```

### 6.3 一票否决

以下情况直接排除，不管得分多高：

- 板块属于一日游，无持续产业逻辑
- 公司近期有大比例减持公告
- 公司被证监会/交易所监管问询
- 分时尾盘偷袭涨停（非真实买盘推动）
- 20 日涨超 100%，严重透支预期

### 6.4 主线题材的人工裁定

`theme_analyzer.py` 给出六维评分排名，但最终主线是由以下因素综合判断的：

1. **技术面数据**（系统给出）— 涨停占比、持续性、资金流入等
2. **政策面判断**（人工）— 板块有无国家级政策支持
3. **产业逻辑**（人工）— 上涨是概念炒作还是产业趋势
4. **与历史规律比对**（工具辅助）— `/backtest` 验证板块历史胜率

**两者分工清晰，互相印证，不互相替代。**

## 七、同板块横向对比

`smart_screener.py` 内置的 `rank_within_sector()` 函数，在输出全局排名后，自动按板块分组进行组内横向对比。

### 7.1 板块内五维排名

| 维度 | 权重 | 说明 |
|------|------|------|
| 均线多头质量 | 35% | 综合评分作为均线代理 |
| 量能强度 | 25% | 是否放量（从评分理由中提取） |

## 九、逻辑验证

`logic_validator.py` 对候选股进行结构化的上涨逻辑验证，回答"为什么涨"和"有没有硬伤"两个问题。

```
    ✅/⚠️/❌     ✅/⚠️/❌     ✅/⚠️/❌
```

| 质押风险 | 🟡 关注 | 出现"质押""爆仓" |

### 9.3 综合评级

## 十、仓位管理

`position_sizer.py` 基于凯利公式 + 市场情绪调整 + 单票上限约束，输出量化仓位建议和分批建仓计划。

### 10.1 凯利公式

```
f* = (p × b - (1-p)) / b

f*  = 理论仓位比例
p   = 预测胜率（来自模型置信度）
b   = 赔率 = 止盈幅度 / 止损幅度
```

### 10.2 调整系数

| 因子 | 系数范围 | 判定条件 |
|------|---------|---------|
| 市场情绪 | 0.5 ~ 1.3 | 恐慌日1.3 / 亢奋日0.5 / 中性1.0 |
| 主题纯度 | 0.5 ~ 1.0 | 第一主线1.0 / 次主线0.7 / 非主线0.5 |

**最终仓位 = 凯利仓位 × 市场情绪 × 主题纯度，上限 25%**

### 10.3 市场情绪判断

| 情绪 | 触发条件 | 系数 |
|------|---------|------|
| 恐慌 | 跌停>100家 or 上证跌>3% | 1.3 |
| 亢奋 | 涨停>100家 and 上证涨>1% | 0.5 |
| 偏多 | 涨停>50家 and 上证涨>0.5% | 0.8 |
| 偏空 | 上证跌>1% | 0.6 |
| 中性 | 其他 | 1.0 |

### 10.4 分批建仓计划

| 仓位 | 策略 |
|------|------|
| < 15% | 一笔买入（小仓位，不必分散） |
| ≥ 15% | 金字塔建仓：50%试探 + 30%回踩加仓 + 20%突破追加 |

```
  → 排除新闻澄清/立案/暴雷的股票
  → 对候选股用 /check 看分时形态 + 量能
  → 判断：W底回踩 or 死猫反弹？

第五步：执行买入
  → 确定止损位（系统已自动计算）
  → 下单 → 设止损单
  → 添加追踪：python3 position_monitor.py --add CODE PRICE
```

```
  总分30-49 → 密切观察
```
