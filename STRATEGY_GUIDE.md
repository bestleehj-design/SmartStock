# Smart Screener — 全市场智能选股策略

## 概述

整合策略 1（均线收敛）、策略 2（板块跟随）、策略 3（综合评分），新增恐慌日过滤、同板块对比、逻辑验证、止损计算。

**6120 只 A 股全扫描，30 秒出结果。**

---

## 快速开始

```bash
# 基础筛选（主线板块 + TOP20）
python3 smart_screener.py

# 恐慌日模式（优先选抗跌翻红票）
python3 smart_screener.py --panic --top 10

# 指定板块
python3 smart_screener.py --sector PCB --top 10

# 深度模式（连新闻一起查，排除澄清/暴雷）
python3 smart_screener.py --deep --top 10

# 全市场扫描（不限于主线板块）
python3 smart_screener.py --all --top 20
```

## 评分体系

### 四个维度，满分 100

| 维度 | 满分 | 评分逻辑 |
|------|------|---------|
| **均线位置** | 30 | 多头排列 +20，短多 +12，回踩 MA20 +8 |
| **题材归属** | 20 | 在主线板块 +20，其他 +2 |
| **成交量** | 15 | 近5日放量 30% 以上 +10 |
| **恐慌日抗跌** | 25 | 翻红 +25，平收 +18，跌 5% 以内 +8 |

### 扣分项

| 扣分项 | 扣分 | 说明 |
|--------|------|------|
| 20 日涨超 60% | -10 | 追高风险 |
| 连续下跌 3 天 | -8 | 趋势转弱 |
| 5 日跌超 15% | -8 | 短期暴跌 |
| 日成交 < 50 万手 | -10 | 流动性太差 |

## 输出字段说明

| 字段 | 含义 |
|------|------|
| `得分` | 0-100 综合评分 |
| `行业` | 申万行业分类 |
| `板块` | 匹配到的主线板块 |
| `加分` | 具体加分的理由 |
| `风险` | 扣分的风险项 |
| `止损` | 建议止损价（MA20 与 10 日最低的 97%） |
| `风险%` | 从现价到止损的亏损百分比 |

## 完整交易流程（5 步）

```
第一步：盘前
  /morning 看隔夜美股 + 板块风向

第二步：盘中
  smart_screener.py --panic --top 20
  找恐慌日翻红/抗跌的票

第三步：深度筛选
  smart_screener.py --deep --top 10
  排除新闻澄清/暴雷的

第四步：人工把关
  用 /check 看分时形态 + 量能
  判断: W 底 or 死猫反弹？

第五步：执行买入
  确定止损位 → 下单 → 设止损单
```

## 与旧策略对比

| | 旧策略 (command_line_screener) | Smart Screener |
|------|------|------|
| 扫描范围 | 全市场 5552 只 | 主线板块 ~1500 只 |
| 评分维度 | 纯技术面 | 技术 + 题材 + 量能 + 恐慌日抗跌 |
| 止损建议 | 无 | 自动计算 |

| 新闻验证 | 无 | --deep 模式拉新闻排除雷 |
| 速度 | ~5 分钟 | ~30 秒 |
| 结果数量 | 数千只（太多） | 100-200 只（可复盘） |
