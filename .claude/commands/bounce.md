# /bounce — 超跌反弹选股器

三大指数同步大跌日（上证≥2% 科创50≥2.5% 创业板≥3%）筛选有资金承接的超跌票，隔日或隔两日反弹卖出。

**用法:** `/bounce` 或 `/bounce --force` 或 `/bounce --backtest`

## 执行逻辑

### 自动模式（默认）

```
python3 src/skills/bounce_screener.py
```

- 从 `market_index_tbl` 读取今日三大指数涨跌幅
- 检查是否触发阈值（三缺一不触发）
- 触发后：加载全市场日K数据，否决过滤，4因子评分，输出TOP 10

### 强制模式

```
python3 src/skills/bounce_screener.py --force
```

- 忽略触发条件，直接选股

### 回测模式

```
python3 src/skills/bounce_screener.py --backtest
```

- 回溯近4年所有同步大跌日
- 验证4个核心因子对次日反弹的预测力：
  1. 跌幅深度
  2. MA60 位置
  3. 盘中承接
  4. 板块主线
- 因子通过验证（胜率>55%，超额>1%）后才纳入评分

## 否决条件

| 条件 | 说明 |
|------|------|
| 触及跌停 | 今日跌幅 ≤ -9.8% |
| 跌破MA60 | 收盘 < MA60（趋势已破） |
| 放量破位 | 成交量 > 3倍均量 且 跌幅 > 8% |

## 因子评分（100分制）

| 因子 | 分值 | 说明 |
|------|------|------|
| 跌幅深度 | 25 | 深跌-5~-8%: +10, 中跌-3~-5%: +5, 回踩MA20支撑: +10, RSI6超卖: +5 |
| 趋势基础 | 25 | 站上MA60: +12, 年线多头: +8, 非加速下跌: +5 |
| 盘中承接 | 25 | 长下影: +15, V型反弹收盘偏上: +10 |
| 板块/资金 | 15 | 主线板块: +8, 白名单联动: +7 |
| 加分项 | 10 | 白名单龙头: +5, MA20附近弹性: +5 |

## 操作建议

- 目标：2-5% 反弹
- 止损：今日最低价
- 仓位：≤ 20%
- 持有周期：1-3 天

## 设计原则

1. **先回测再编码** — 仅纳入通过回测验证的因子
2. **K线形态不直接用** — 量化资金常见下影线诱多，用资金端代理指标
3. **触发阈值严格化** — 上证≥2%、科创≥2.5%、创业板≥3%（三缺一不触发）
4. **回测成功标准** — 胜率 > 55%、超额 > 1%

## 数据来源

- `market_index_tbl` — 三大指数日K
- `daily_info_tbl` — 个股日K
- `stock_basic_info_tbl` — 行业分类/股票元数据
