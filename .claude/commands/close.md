# /close — 盘后综合分析

收盘后结合大盘、持仓、量价关系，做明日趋势预判和操作建议。

**用法:** `/close`

**执行逻辑:**
1. 运行 `python3 close_analysis.py` 获取全量数据
2. 大盘：三指数涨跌幅 + 量比 + 涨跌家数
3. 结构判断：放量/缩量 × 上涨/下跌 → 四种组合
4. 持仓：今日表现 + 量能 + MA位置 + 日内形态
5. 结合 SOX 和宏观事件给出明日确认条件
6. 逐只输出明日操作建议

**数据来源:** 新浪日K + akshare + 本地 MySQL

**依赖:**
- 需在境内网络或 VPN 环境（akshare 涨跌家数）
- SOX 数据需已同步到 MySQL sox_index_tbl
