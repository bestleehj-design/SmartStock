# /morning — 每日盘前综合分析

运行盘前综合分析报告，自动拉取隔夜美股、港股走势、板块轮动、持仓新闻、T+0 交易计划。

**用法:** `/morning`

**可选:** `/morning 2026-06-05` — 分析指定日期

**执行逻辑:**
1. 运行 `python3 src/skills/recall_log.py` 回调昨日判断（加载上次的决策和逻辑）
2. 运行 `python3 src/skills/morning_analysis.py` 获取全量数据
3. 阅读 `trading_plan.md` 获取最新持仓和操作计划
4. **国际要闻搜索与持仓传导分析** — 搜索隔夜全球要闻 / 美股涨跌原因，AI 分析对 A 股和持仓的传导路径
5. **置信度标注** — 报告中每条判断标注置信度级别（高/中/低，附依据）
6. 总结关键结论：美股/港股方向、板块资金流向、持仓风险信号、综合判断（带置信度标注）
7. 输出长电科技 T+0 操作计划（支撑/阻力/三种情景）
8. 对比昨日判断和今日盘面，标注已验证/证伪的结论

---

### 步骤 4 详细说明：国际要闻与持仓传导分析

**4.1 新闻搜索**
- 使用 WebSearch 搜索 `"隔夜全球市场要闻 / 美股涨跌原因 / what moved markets overnight"`
- 提取 3-5 条核心事件（优先取对 A 股有传导潜力的事件）

**4.2 对每条事件执行 AI 分析**（提取以下维度）：
- a) **事件性质** — 一次性 / 短期(1-3天) / 中期(1-4周) / 长期(1月+)
- b) **对 A 股大盘的传导方向** — 利多 / 利空 / 中性
- c) **对 A 股具体板块的传导** — 映射到消费/科技/能源/军工/金融等板块
- d) **对持仓标的的影响** — 逐只映射（从 `trading_plan.md` 获取当前持仓列表）
- e) **情绪传导路径** — 以链条形式写出路径（例：美伊冲突 → 油价涨 → 成本压力 → 制造业承压）

**4.3 报告输出格式**（作为独立章节直接输出）：
```
──────────────────────────────────────────────────────
  🌐 隔夜国际要闻 & 持仓传导分析
──────────────────────────────────────────────────────

  📰 事件N: {标题摘要}
     性质: {持续时间判断}    压力方向: {对A股整体方向}
     板块映射:
       → {板块1}: {方向+置信度}
       → {板块2}: {方向+置信度}
     持仓影响:
       → {标的1}: {影响判断}
       → {标的2}: {影响判断}
     情绪链: {传导路径链条}
```

### 步骤 5 详细说明：置信度标注

报告中每条判断必须标注置信度级别，使用以下符号体系：

| 符号 | 级别 | 判定标准 |
|------|------|----------|
| ◆ | 高置信度 | 有回测数据支撑（如 SOX→A股传导，898天回测） |
| ◆ | 高置信度 | 来自已验证规则（如持仓分级制，规则1-9） |
| ▲ | 中置信度 | 逻辑推理但未回测（如 CPI 核心低于预期→情绪修复） |
| ▲ | 中置信度 | AI 分析的事件→板块映射 |
| ✕ | 低置信度 | 纯联想，无数据无逻辑链（如 美伊交火→军工股涨） |

**标注规则：**
- 每个判断结论后以 `{符号} {级别} — {依据}` 格式标注
- 依据需简短说明（1句话），让读者知道置信度来源
- 如果一条判断包含多个子判断，各自标注

**在以下章节中应用置信度标注：**
- 美股/港股方向判断
- 板块资金流向分析
- 持仓新闻情绪判断
- 综合判断（多空信号计数）
- T+0 操作计划的支撑/阻力位判断
- 国际要闻与持仓传导分析（每条映射标注）
