# /check — 单票快速分析

分析指定股票的今日分时、技术指标、近期日K、均线、财务数据、新闻舆情。

**用法:** `/check <股票代码>` 或 `/check <股票名称>`

**示例:**
- `/check 600584` — 分析长电科技
- `/check 顺络电子` — 按名称搜索分析
- `/check 00981 HK` — 分析港股中芯国际

**执行逻辑:**
1. 如果用户给的是名称，先通过 `akshare.stock_info_a_code_name()` 查找代码
2. 运行 `python3 stock_check.py <代码>` 获取完整数据
3. 结合数据给出技术面分析、止损/止盈建议

**数据来源:** 东方财富 + 新浪财经 + 本地数据库
