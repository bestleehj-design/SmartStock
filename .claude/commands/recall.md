# /recall — 回调昨日判断

加载上一次分析的结论，用于新会话开场时回顾历史。

**用法:** `/recall` — 加载昨天的判断

**可选:** `/recall 2026-06-08` — 加载指定日期的判断

**执行逻辑:**
1. 运行 `python3 recall_log.py [日期]`
2. 按操作类型分组展示（卖出/持有/等待/买入）
3. 显示每条的逻辑、风险、止损/目标价
4. 结合当天的实际涨跌，标注哪些判断已被验证/证伪

**数据来源:** MySQL `claude_trades` 表

**何时使用:**
- 每天开盘前，配合 `/morning` 使用
- 新会话开始时，了解上次做了什么决策
